📌 Smart Shopping Assistant

A web-based intelligent shopping assistant that helps users select the best combination of products within a given budget using the 0/1 Knapsack Algorithm (Dynamic Programming).

🚀 Features
🛒 Add & remove products
🎯 Smart product selection using Knapsack Algorithm
💰 Budget slider for filtering products
🎙️ Voice input for budget (optional)
🌙 Dark / Light mode toggle
📂 Category filter
🔐 User Login & Registration system (SQLite)
🖼️ Product images support
🧠 Algorithm Used

This project uses the 0/1 Knapsack Algorithm (Dynamic Programming) to:

Maximize total product value
Stay within user's budget
Suggest best combination of items
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Python Flask
Database: SQLite (users.db)
Data Storage: JSON (products.json)
